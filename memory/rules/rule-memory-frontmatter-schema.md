# Memory Frontmatter Schema Rule

Every memory note should include these fields:

1. `id`
2. `title`
3. `type`
4. `tags`
5. `created`
6. `updated`
7. `scope`
8. `status`

Use `[[note-id]]` links for durable cross-note navigation.
